How can I prepare for college?

Many helpful learning websites can help future computer science majors prepare to enter college. We have created a list of websites that can help students learn independently about various topics in computer science to aid them in preparing for their degree program.

Number 1: The Odin Project

The Odin Project allows aspiring web developers to learn, all for free. Their courses include HTML, CSS, JS, NodeJS, and Ruby



Number 2: Codecademy

Codecademy is a website that allows students to independently learn how to code in various programming languages.



Number 3: Khan Academy

Khan Academy is a free website that offers lessons in a plethora of subjects to students, including mathematics and programming.



Number 4: TryHackMe

TryHackMe is a website that provides free resources for students wishing to get started in the field of cybersecurity.



Number 5: HackTheBox

HackTheBox is a website that provides cybersecurity lessons, challenges, and more for passionate students.



Number 6: Code.org

Code.org is a website for new programmers to learn and practice their coding skills.



Number 7: edX

edX is a website with free college level courses in a variety of subjects, including programming.



Number 8: Edube

Edube is a website that contains lessons and certifications for various programming languages.
